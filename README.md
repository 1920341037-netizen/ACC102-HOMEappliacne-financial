# ACC102 Python Data Product | Track 2 GitHub Data Analysis Project
## 🎯 Project Overview
This project conducts a comprehensive data analysis of **Apple Inc. (AAPL) stock performance in 2024**, covering price trends, volatility, trading volume, and key technical indicators. The analysis aims to provide clear, data-driven insights for finance beginners and accounting students to understand Apple's 2024 stock market performance and its underlying business drivers.

## 📈 Project Objectives
1.  Analyze Apple's 2024 stock price movement, including daily open/high/low/close prices and overall annual return
2.  Evaluate stock volatility and risk through metrics like daily returns, standard deviation, and maximum drawdown
3.  Examine the correlation between stock price and trading volume to identify market sentiment trends
4.  Calculate and visualize key technical indicators (e.g., moving averages, RSI) to assess price momentum
5.  Summarize actionable financial insights for investors and accounting learners

## 📊 Data Source
- **Data Provider**: Yahoo Finance (via `yfinance` Python library)
- **Stock Ticker**: AAPL (Apple Inc.)
- **Time Range**: January 1, 2024 – December 31, 2024
- **Data Fields**: Date, Open, High, Low, Close, Adjusted Close, Volume
- **Data Preprocessing**: Handled missing values, filtered 2024 trading days, calculated derived metrics (daily returns, moving averages)

## 🛠️ Tools & Libraries Used
- **Programming Language**: Python 3.x
- **Core Libraries**:
  - `pandas`: Data loading, cleaning, and statistical analysis
  - `numpy`: Numerical calculations (returns, volatility, etc.)
  - `yfinance`: Real-time stock data acquisition
  - `matplotlib` / `seaborn` / `plotly`: Data visualization (price charts, volume trends, technical indicators)
  - chatgpt4.0 and Grok for code error correction
- **Environment**: Jupyter Notebook
